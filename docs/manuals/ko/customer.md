# 고객 매뉴얼 — Seoul Aqua 고객 포털

> 본 매뉴얼은 2026-06-01 sprint 기준. CustomerContact (계약 주체 + 관리 주체) 가 사용할 수 있는 고객 포털 가이드입니다.

## 1. 로그인

- URL: `https://<도메인>/ko/portal/login` (고객 전용)
- ID: 등록된 전화번호
- 비밀번호:
  - 계약 활성화 또는 판매 완료 시 자동으로 10자 임시 비밀번호가 SMS 로 발송됩니다 (`SMS_PORTAL_WELCOME`)
  - 첫 로그인 시 새 비밀번호 강제 설정

⚠️ **이 페이지는 고객 전용** — 본사 직원/현장 직원 ID 로 시도해도 작동하지 않습니다. 직원이 본인 ID 로 시도 시 다른 그룹의 로그인 페이지로 안내됩니다 (phase 2 예정).

### 비밀번호 분실 시
회사 또는 담당 기사에게 문의 → MANAGER 이상이 비밀번호 리셋 → 새 임시 비밀번호 SMS 발송.

## 2. 포털 메뉴

- **대시보드**: 보유 장비, 다음 방문, 미수금 요약
- **장비**: 설치된 정수기/공기청정기 목록, 모델/설치일/필터 교체 이력
- **계약**: 계약서 다운로드 (PDF), 부록 이력
- **방문**: 과거 방문 + 작업확인서 PDF + 예정 방문
- **서비스 요청**: 점검/수리/교체/이전 신청
- **수금**: 영수증 (PDF), 미수금
- **세금계산서**: B2B 한정 — e-tax invoice 이메일 보관함
- **담당자 관리** (계약 주체 한정): 관리 주체(OPS) 추가/삭제, 본인 정보 수정
- **설정**: 언어, 비밀번호 변경, 알림 수신 설정 (per-channel opt-out)

## 3. 서비스 요청 흐름

1. "서비스 요청" 메뉴 → "새 요청" 버튼
2. 유형 선택: 정기 점검(무상) / 수리(유상) / 교체 / 이전
3. 희망 일정 + 비고 입력
4. 제출
5. 회사 검토 후 SMS/이메일로 결과 안내:
   - **무상 유형**: 자동 승인 + Visit 자동 생성
   - **유상 유형**: 사무실에서 검토 후 승인/거부

## 4. 영수증 PDF 형태 (PR #11 신규)

기사가 현장에서 수금 후 받은 영수증은 다음 형태:
- A4 한 장에 동일 영수증 2부 + 가운데 절취선
- 가위 그림 따라 자른 후 한 부 보관 (한 부는 회사 보관)
- 금액 칸에 ₫ 단위만 인쇄, 기사가 펜으로 실제 수금액 손기입
- 결제 방법은 한국어/베트남어 병기 (예: "은행 이체 / Chuyển khoản")

⚠️ 현장에서 청구액이 사전 예상과 달라질 수 있습니다 (예: 추가 부품, 부분 수리, 영업 할인). 기사가 새 금액으로 변경할 경우 영수증 PDF 가 새 값으로 자동 재생성됩니다.

## 5. 알림

- **계약/세금계산서/법적 통보**: 계약 주체에게 (해당 언어로)
- **방문 SMS/영수증/일정**: 주 OPS 담당자에게 (해당 언어로)
- **미수금 독촉**: 계약 주체 + 모든 OPS CC

수신 채널은 설정 → 알림에서 SMS / 이메일 각각 opt-out 가능. 단, **보안 메시지(비밀번호 리셋, 영수증)** 는 항상 발송 (opt-out 무시).

## 6. 알려진 제약

- 포털은 mobile-first PWA — 데스크탑에서도 사용 가능하나 모바일 최적화
- URL: 현재 `/ko/portal/...` 형태, phase 2 에서 root path `/ko/...` 로 이동 예정 (단축 URL 효과 — SMS 비용 절감)
- 본사 직원/기사 페이지 진입 시도 시 거부됨 (cross-realm 차단)
